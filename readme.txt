Estos son las carpetas que contiene todo lo pedido por el proyecto

Carpetas:

-> CalDepGUIFuentes
    Esta carpeta incluye los archivos fuente de la implementación gráfica, estos son:

    - app.py: Este es el archivo de la interfaz el cual debe ser ejecutado
    - CalendarioDeportivo.mzn: Este es el archivo del modelo genérico creado por nosotros
    - DatosCalendarioDeportivo.dzn: Este es el archivo de los datos generados al ingresarle los datos por medio del txt
    - requirements.txt: Este es el archivo que contiene los nombres de lo que debemos decargar para que la app se ejecute bien

-> DatosCalDep
    Esta carpeta incluye los datos con que fue probado nuestro modelo, estos datos estan en archivos de tipo txt

-> MisInstancias
    Esta carpeta incluye las 5 instancias generadas por nuestro equipo de trabajo para retar a otros proyectos que resuelvan el mismo problema


Intrucciones:

1. Abrir la carpeta 2025301-2025513-2040430-2040506-Proyecto2-AdaII.2023-I desde la terminal con el comando: code .
2. Despues debe abrir la terminal del visual studio y poner el comando: cd .\CalDepGUIFuentes\ para ir a la carpeta de las fuentes 
3. Despues para descargar lo necesario para ejecutar la app debe poner el comando: pip install -r requirements.txt   
4. Finalmente se debe ubicar en el archivo app.py y ejecutarlo
